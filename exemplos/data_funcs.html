returns
  • data_funcs.get_days_between(dataMenor, dataMaior) data formato dd/mm/aaaa => retorna intDias entre as datas. ex: get_days_between('01/01/2000', '10/01/2000') = return 9
  • data_funcs.compara_datas(dataIni, dataFim) data formato dd/mm/aaaa => retorna boolean se dataIni > dataFim
  • data_funcs.is_data_valida(data) data formato dd/mm/aaaa => retorna boolean se data informada é correta (checa inclusive se dia do mês está correto (mês de 30,31,fev,bisexto,etc)
  • data_funcs.data_split(data, intStart, intEnd) data formato dd/mm/aaaa => parte a data. return intValor. ex: pegar o dia: data_split(data, 0, 2), pegar o mes: data_split(data, 2, 2), pegar o ano: data_split(data, 4, 4)
  • data_funcs.is_max_value_for(strItem, intValor) => retorna boolean se valor estiver dentro dos limites conforme o item (item pode ser: hora, minuto, segundo, dia, mes ou ano). ex: hora: true se valor entre 0 e 24, minuto e segundo: true se entre 0 e 59, ano: entre 1970 e 2038
  • data_funcs.get_last_day_of(intMes, intAno) ano formato aaaa retorna intDia conforme o mes/ano informado. ex: get_last_day_of(2, 2012) retorna 29 (fev.bisexto)
