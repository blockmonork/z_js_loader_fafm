jquery.gramaptcha

Simples captcha usando sequência definida pelo usuário

$(elemento).gramaptcha(defaultOptions{
	inputId:'gramaptcha',
	hiddenId:'v',
	outerBoxClass:'',
	isDebug:false,
	theItens: ['Eu', 'Tu', 'Ele', 'Nós', 'Vós', 'Eles'],
});
Onde:
inputId = o elemento (form.input) que vai receber o valor digitado pelo usuário.
hiddenId = a resposta da sequência
outerBoxClass = classe css para estilização do box que exibe a sequência
isDebug = se true apenas exibe um alert com o resultado gerado
theItens = array com a sequência a ser gerada

ex. formulário de contato:
Nome: | Email: Complete a sequência abaixo:

num submit desse formulário, bastaria fazer a checagem: if (gramaptcha.value == v.value){true}else{false}
